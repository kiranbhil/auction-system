# Real-Time Auction System
This is a simple real-time auction system that allows users to place bids and view a live leaderboard of the top 5 bidders. The application is built using the MERN stack (MongoDB, Express, React, Node.js) along with Socket.IO for real-time updates.

## Features
<h5>Users can place bids in real-time.</h5>
<h5>Users can view a live leaderboard of the top 5 bidders.</h5>
<h5>Real-time updates are handled using Socket.IO.</h5>

## Technologies Used
<h5>MongoDB</h5>
<h5>Express</h5>
<h5>React</h5>
<h5>Node.js</h5>
<h5>Socket.IO</h5>


# Installation
<h5> 1). Clone this repository onto your local machine. </h5>
<link>https://github.com/kiranbhil/auction-system.git</link>
<h5> 2). Install the dependencies for Backend. </h5>
<h5>cd backend </h5>
<h5>npm install</h5>
<br/>
<h5> 3). Install the dependencies for Frontend. </h5>
<h5>cd frontend </h5>
<h5>npm install</h5>
<br/>
<h5> 4). Start the server. </h5>
<h5>npm run server </h5>
<h4>The server should now be running on <link>http://localhost:8000</link>. </h4>

<h5> 5). Start the client. </h5>
<h5>npm start </h5>
<h4>The application should now be running on http://localhost:3000. </h4>
